Ory Kratos 是世界上第一个云原生身份和用户管理系统

kratos ory 开源的新oauth2 方案（api 优先方案）

包含的特性

- 自服务的登录以及注册
- MFA/2FA 支持，支持totp 协议
- 账户校验
- 账户恢复
- 个人信息管理
- admin api
