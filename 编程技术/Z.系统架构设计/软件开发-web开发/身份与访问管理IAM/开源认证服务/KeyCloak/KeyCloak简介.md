Keycloak 是一款开源的身份和访问控制软件，提供单点登录功能，以及可自定义的用户界面，用于登录、注册和帐户管理等。您可将其集成到现有的 LDAP 和 Azure Active Directory 服务器中，也可以将身份验证委派给第三方身份提供商

该解决方案主要包括以下特点：

- 单点登录: 支持 OpenID Connect、OAuth 2.0 和 SAML 2.0 标准协议
- 身份和访问管理: 提供用户联合、强认证、用户管理、细粒度授权等。轻松的添加身份验证到应用程序和服务中，而无需处理存储用户或验证用户
